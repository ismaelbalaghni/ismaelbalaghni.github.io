Stocksinator...

...ou comment découvrir la programmation mobile avec un projet amusant

Dépôt GitHub

L'objectif de ce cours était de découvrir la base de la programmation mobile sur Android.

Les autres objectifs étaient aussi de découvrir les bases de Git (qu'on aura revues dans d'autres cours) et tout ce qui est design pattern.

Mon idée de base était de faire une application listant les chefs d'états des pays et afficher une sorte de présentation de chacun d'entre eux. Mon délire allait même plus loin en permettant de récupérer tous les chefs d'état à travers l'histoire.

MAIS mon idée était simplement impossible. Porte de sortie : on liste toutes les entreprises et autres indices listés en bourse (via FinancialModelingPrep) et permettre à l'utilisateur de voir le détail de chaque entreprise ou indice.

Vous aurez plus d'infos sur GitHub !
